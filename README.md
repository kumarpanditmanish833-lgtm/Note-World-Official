## 🌟 About

**NOTs-WORLD** is a simple, student-friendly web platform that provides easy access to semester-wise notes for Computer Science students. All notes are organized neatly by semester so you can find exactly what you need, fast.

---

## ✨ Features

- 📂 **Semester-wise Organization** — Notes neatly grouped from Semester 1 to Semester 6
- 🔗 **Direct PDF Links** — One-click access to study materials and syllabi
- 🎨 **Clean UI** — Dark navy theme with smooth accordion-style subject dropdowns
- 📱 **Lightweight** — Pure HTML & CSS, no heavy frameworks

## 🚀 Getting Started

No installation needed! Just open the project in your browser:

```bash
# Clone the repository
git clone https://github.com/kumarpanditmanish833-lgtm/NOTs-WORLD.git

# Navigate into the project
cd NOTs-WORLD

# Open index.html in your browser
open index.html   # macOS
start index.html  # Windows
xdg-open index.html  # Linux
```

Contributions are welcome! If you'd like to add notes for missing semesters or subjects:
